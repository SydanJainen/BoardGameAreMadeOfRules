SITUATION ANALYSIS:
The player is in the early game with only 4 tiles placed (2 mountain, 1 forest, 1 wetland) out of the 20 turns total. Animals placed so far are 1 bear on a mountain tile and 1 salmon on the wetland tile. The player holds 2 nature tokens, which provide flexibility to decouple tile and token selection.

Key scoring objectives in play:
- Bears want pairs: The player already has 1 bear placed and needs exactly 1 more adjacent bear to complete a pair. Bear Card A scores an increasing number of points based on total pairs (4 points for 1 pair, 11 for 2 pairs, 19 for 3 pairs). Completing the first pair early is valuable and sets up the path toward additional pairs later.
- Salmon want runs: 1 salmon is already placed on the wetland. Runs score based on length (a run of 2 scores 2, run of 3 scores 4, run of 4 scores 8, etc.). Building toward longer runs is critical, but there is time since only 1 salmon is down.
- Hawks want line-of-sight: No hawks are placed yet. Hawks score for straight-line hexagonal connections between hawks, so they benefit from being spread across the map rather than clustered.

The market offers:
- Tiles: forest+river, mountain+prairie, wetland+forest
- Tokens: hawk, elk, bear, fox

The mountain+prairie tile paired with the bear token is the critical combination to evaluate, since the player needs a bear adjacent to the existing bear for the pair bonus.

RECOMMENDED MOVE:
Spend 1 nature token to take the mountain+prairie tile AND the bear token (decoupling tile and token selection). Place the mountain+prairie tile adjacent to one of the existing mountain tiles (extending the mountain corridor and providing adjacency to the current bear). Place the bear token on the new mountain+prairie tile, adjacent to the existing bear, completing the bear pair.

REASONING:
1. Completing the bear pair is the highest-priority immediate objective. Bear pairs that are completed early free the player to focus on other scoring goals in subsequent turns. With Bear Card A, the first pair alone is worth 4 points, and each additional pair increases the total score significantly (11 for 2 pairs, 19 for 3 pairs).

2. The mountain+prairie tile extends the mountain habitat corridor. With 2 mountains already placed, adding a third connected mountain tile builds toward a larger corridor for end-game habitat scoring (1 point per tile in the largest contiguous group of each habitat type, plus potential majority bonus).

3. The mountain+prairie tile is a dual-terrain tile, which adds prairie to the player's environment. Since the player currently has no prairie tiles, this opens a new habitat type, which is useful for both wildlife placement flexibility and habitat diversity scoring.

4. Spending 1 nature token is justified because the default pairings in the market may not offer the mountain+prairie tile paired with the bear token. Using a nature token to select any tile and any token independently ensures the optimal combination. The player still retains 1 nature token (worth 1 point at end-game or usable for future flexibility), and the strategic value of completing the bear pair far exceeds the 1-point cost of the spent token.

5. Placement specifics: The mountain+prairie tile should be placed so that the mountain edge is adjacent to the existing mountain tile that holds the bear. This ensures the bear can legally be placed on the new tile (bears must be among the wildlife options shown on the tile, and mountain+prairie tiles typically show bear or fox), and the new bear will be directly adjacent to the existing bear, forming the scoring pair.

ALTERNATIVE MOVES:
1. Take the wetland+forest tile and the bear token (spending a nature token to decouple). Place the wetland+forest tile adjacent to the existing wetland to extend that corridor, then place the bear on any available mountain tile that is adjacent to the existing bear. This prioritizes wetland corridor growth alongside the bear pair. However, this only works if there is an existing open mountain tile adjacent to the current bear that accepts bears. Since no such open tile is confirmed, this is riskier than placing a new mountain tile specifically for the bear.

2. Take the mountain+prairie tile and the hawk token as a natural pair (no nature token spent). Place the mountain+prairie tile to extend the mountain corridor and place the hawk on any legal tile to start building toward hawk line-of-sight scoring. This preserves both nature tokens but sacrifices the immediate bear pair completion. The bear pair would need to wait for a future turn when another bear token appears in the market, which is uncertain and delays a key scoring objective.

3. Take the forest+river tile and the bear token (spending a nature token to decouple). Place the forest+river tile to start building river and forest corridors, then place the bear on an existing open tile adjacent to the current bear. This diversifies habitat types (adding river) but is suboptimal because it does not add a new mountain tile, and the player needs to confirm an open adjacent tile for the bear. The mountain+prairie option is more reliable for guaranteeing the bear pair placement.
